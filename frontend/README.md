# 前端项目 (Vue 3)

## 项目结构

```
src/
├── main.ts          # 应用入口
├── App.vue          # 根组件
├── components/      # 公共组件
├── views/           # 页面组件
├── router/          # 路由配置
├── store/           # 状态管理
└── assets/          # 静态资源
```

## 安装依赖

```bash
npm install
```

## 开发运行

```bash
npm run dev
```

## 构建生产版本

```bash
npm run build
```

## 技术栈

- Vue 3 + TypeScript
- Vue Router 4
- Pinia (状态管理)
- Element Plus (UI组件库)
- Vite (构建工具)
